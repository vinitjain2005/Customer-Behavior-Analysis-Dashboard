## 📌 Overview

The Customer Behavior Analysis Dashboard is a data analytics project that analyzes customer shopping patterns, purchasing behavior, product preferences, and customer engagement using **Python, MySQL, and Power BI**.

The project demonstrates the complete analytics workflow from data cleaning and exploratory analysis to database integration and interactive dashboard creation.

---

## 🚀 Technologies Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* MySQL
* Power BI
* Jupyter Notebook

---

## 📂 Dataset

Dataset File:

* `customer_shopping_behavior.csv`

Key attributes include:

* Customer Demographics
* Product Categories
* Purchase Amount
* Shopping Frequency
* Payment Methods
* Review Ratings
* Seasonal Purchases
* Subscription Status

---

## 🔄 Project Workflow

1. Data Cleaning and Preprocessing using Python
2. Exploratory Data Analysis (EDA)
3. Data Storage and Querying with MySQL
4. Dashboard Development in Power BI
5. Business Insights Generation
